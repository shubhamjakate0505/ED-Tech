
# 📘 StudyNotion – EdTech Platform  

## 🚀 Overview  
StudyNotion is a full-stack EdTech platform built using the MERN stack that enables users to create, consume, and rate educational content.  

It provides a seamless experience for students to learn and instructors to teach and manage courses.

---

## 🛠️ Tech Stack  

- Frontend: React.js, Redux, Tailwind CSS  
- Backend: Node.js, Express.js  
- Database: MongoDB  
- Other Tools: JWT, Bcrypt, Cloudinary, Razorpay  

---

## ✨ Key Features  

- Authentication (Login/Signup, OTP, Forgot Password)  
- Course browsing and enrollment  
- Wishlist & Cart functionality  
- Secure payments using Razorpay  
- Video and content management via Cloudinary  
- Course rating and reviews  
- Instructor dashboard for course management  

---

## 🏗️ Architecture  

- Client-Server architecture  
- REST API-based communication  
- JWT-based authentication
